# BOM
This is a of parts you will need to build the PCBs for the keyboard yourself. These parts are examples and if you have suitable replacements, you can use them.

Both the Button and Slide Switches are technically optional. The Button is used for resetting the board. When not connecting the power switch, short the power jumper with some solder.
